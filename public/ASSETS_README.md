# Public Assets Directory

## Logo File

Place your `1AG_Color_Web.png` logo file in this directory.

The logo should be:
- Named exactly: `1AG_Color_Web.png`
- PNG format with transparent background (or black background that will blend)
- Recommended size: At least 200px height for best quality

## Other Assets

You can also place other static assets here:
- `favicon.png` - Your site favicon
- Additional images or documents you want publicly accessible

All files in this directory will be copied to the root of your deployed site.
